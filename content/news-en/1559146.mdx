Ubuntu 26.10 is codenamed "Stonking Stingray." As this is not an LTS version, its lifecycle is only 9 months. These non-LTS versions typically offer rolling updates, allowing users to gain access to new features and improvements more quickly.

Based on the timeline, Ubuntu 26.10 is likely to adopt the Linux Kernel 7.2 by default, which is expected to be released in late August 2026. Ubuntu 26.10 will freeze the kernel by October 1st, so adopting Linux Kernel 7.2 is perfectly feasible.

Here is the release plan for Ubuntu 26.10:

April 23, 2026: Ubuntu 26.04 LTS version released, 26.10 version will be developed based on this version.

April 30, 2026: Ubuntu 26.10 development officially begins.

August 20, 2026: Ubuntu 26.10 enters feature freeze.

September 24, 2026: Ubuntu 26.10 is opened for public testing.

October 1, 2026: Ubuntu 26.10 enters kernel freeze.

October 8, 2026: Ubuntu 26.10 enters final freeze.

October 15, 2026: Ubuntu 26.10 official version released.

June 2027: Ubuntu 26.10 reaches end of life and will no longer provide software update support.
