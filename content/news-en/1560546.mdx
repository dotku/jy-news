When you browse webpages, open files, or review knowledge bases or notes, Copilot will hover in a floating window within the IMA application, automatically sensing the current content without requiring you to upload files.

For example, simply asking "What is this webpage about?" will immediately provide an answer based on the current page. On knowledge base or note pages, you can also directly instruct it: "Help me organize the content of this webpage into notes and add it to my knowledge base."

Copilot includes official skill packs (Skills), with the initial release including: knowledge base operations, note operations, Skill creation, and report generation, all deeply integrated with IMA products. Compared to previous versions, the knowledge base Skill now supports reading file content, enabling more complex cross-file information reading and summarization.

You can also command it using natural language, such as "Help me categorize this knowledge base by subject." In addition to official skills, users can also freely add custom Skills. Furthermore, Copilot supports configuring API Keys for various models, providing powerful expansion capabilities for advanced users (using your own API does not consume platform computing power).

Copilot is now available on Mac, Windows, iOS, Android, and HarmonyOS platforms. To ensure user experience, the function is implemented through an application system and will be gradually released in order of application. New users will receive a computing power benefit, and additional computing power can be obtained by logging in daily.

IMA Copilot, through personalized memory, scene perception, skill ecosystem, and anthropomorphic interaction, is committed to becoming a truly understanding, accompanying, and assisting personal knowledge partner for handling various tasks. Whether you are a student, researcher, or content creator, you can find new possibilities for improving efficiency and creativity within it.
