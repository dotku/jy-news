The user confirmed this anomaly with a benchmark test using CrystalDiskMark. The official specifications for the Samsung 970 Evo list sequential read speeds of 3400 MB/s and write speeds of 2500 MB/s, but the measured data after the upgrade was only 26% and 34% of those values, respectively.

However, random read/write performance did not show significant changes, likely because the test load did not fully utilize the SLC cache.
