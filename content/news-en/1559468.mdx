Currently, eligible iPhone and iPad devices can directly download and install the new system version online through “Settings → General → Software Update”. According to Apple’s description in the release notes, this update includes several unspecified bug fixes and security updates, aiming to enhance the overall user experience and system protection capabilities.

In addition to launching iOS 26.4.2 and iPadOS 26.4.2 for the latest system devices, Apple also simultaneously released iOS 18.7.8, for older iPhone models that do not support upgrading to iOS 26, to continue providing necessary security maintenance for users still running older system versions.

While continuing to release minor version fixes, Apple has already begun pushing iOS 26.5 and iPadOS 26.5 to developers and beta testers. At the current pace, this new version is expected to be officially released to the public in late May, further bringing functional improvements and experience optimizations to iPhone and iPad.
