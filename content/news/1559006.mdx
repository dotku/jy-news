在分屏浏览方面，Firefox 150 对 Split View 进行了明显增强，用户现在可以在任意链接上点击右键，直接选择“在分屏视图中打开链接”，让新页面与当前标签并排显示，便于对照阅读或多任务处理。创建分屏视图时，用户还可以搜索已打开标签页，并通过标签页右键菜单中新增的“反转标签”选项，迅速交换左右两侧标签的位置。

多标签分享功能也得到强化。用户可以一次性选中多个标签页，右键选择“分享 X 个链接”，随后将生成的内容粘贴到其他应用中时，每个链接都会同时包含网页标题和 URL，方便在邮件、聊天或文档中清晰展示所分享的页面列表。

内置 PDF 编辑器是本次更新的另一个重点。Firefox 150 允许用户对 PDF 文档页面进行重新排序、复制、粘贴、删除，并支持按页面导出等操作，使浏览器本身就能满足更复杂的 PDF 管理与编辑需求，减少对外部工具的依赖。

在安全和隐私层面，Firefox 将此前仅对“增强型跟踪保护”设为“严格”的用户开放的本地网络访问限制，扩展至所有用户，并采取渐进式推送。未来，当网站尝试访问本地网络中的设备，或连接到用户设备上的应用与服务时，浏览器将强制要求用户授权，防止网页在用户不知情的情况下扫描或利用局域网资源。

翻译功能方面，Mozilla 继续推进本地、实时、隐私友好的翻译方案。用户可在地址栏输入“translate”，快速调出 about:translations 页面，体验无需将内容上传云端的实时翻译能力。新版中相应界面与功能也得到更新，以提升使用便捷性。

针对不同平台，Firefox 150 也加入了多项定制改进。Linux 用户现在可以使用 GTK 系统 Emoji 选取器，通过系统快捷键（通常为 Ctrl+.）在浏览器中插入表情符号，与桌面环境体验保持一致。对于通过微软商店安装 Firefox 的 Windows 用户，浏览器 Web 应用（PWA）现已正式可用，便于将常用网站以“应用”形式固定和启动。

配置文件管理系统在本版中全面向用户开放，覆盖 Windows 10 在内的更多平台。Firefox 150 允许 Windows 10 和 Windows 11 用户将浏览器配置文件备份为单独文件，这一功能同样适用于采用新配置文件管理系统的用户，有利于迁移、恢复或在多设备间同步浏览环境。

Linux 发行版方面，Mozilla 为基于 RPM 的系统（如 Red Hat、Fedora、openSUSE 等）提供了新的 .rpm 安装包，以便这些发行版的用户更方便地获取和部署最新版 Firefox。此外，对于习惯通过右键菜单管理标签组的用户，Firefox 新增了相关选项，允许在设置中关闭“拖拽标签自动创建标签组”的行为，路径为“设置 > 标签页 > 拖动标签以创建标签组”。

目前，Firefox 150.0 已通过 Mozilla 的 FTP 服务器提供下载，现有用户将在接下来数小时内陆续收到更新推送：

https://ftp.mozilla.org/pub/firefox/releases/150.0/
