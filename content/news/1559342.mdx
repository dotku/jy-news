目前 A 社主要提供每月 20 美元的 Claude Pro 订阅和每月 100 美元起步的 Claude Max 订阅，原本这些订阅都包含全部付费权益，只不过 Max 订阅可以获得更高的 Claude Code 使用配额 (5x 或 20x)。

所以如果 20 美元的订阅删除 Claude Code 权益的话，开发者就至少需要支付每月 100 美元来开通 Claude Max 订阅，假如开发者用量没有那么高，订阅 100 美元版的 Claude Max 就会浪费。

针对这个消息 A 社工程师也进行回应，简单来说现在的情况是 A 社正在悄咪咪的进行 A/B 测试，大约有 2% 的新准备开通 Claude Pro 订阅的用户将无法使用 Claude Code，其余 98% 准备新开通的用户可以正常使用 Claude Code。

而现有的 Claude Pro 和 Claude Max 订阅用户也都可以继续使用 Claude Code，相关权益完全不受影响，至于为什么如此测试，大概率是 A 社想要逼迫开发者们支付更高的费用才能使用 Claude Code。

如何应对这种区别对待：如果开发者实在要开通 Claude Pro 并使用 Claude Code，则在开通时应当仔细检查权益，如果发现不包含 Claude Code 则可以考虑换 IP + 注册新账号进行测试，以免自己的付费权益受到影响。
